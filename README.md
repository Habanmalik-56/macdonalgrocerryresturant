# McDonald Grocery & Restaurant Website

This is a complete, responsive 5-page website for "McDonald Grocery & Restaurant", built using HTML5, CSS3, and Vanilla JavaScript.

## 🚀 Features
- **Design:** Warm Organic Small-Town Vibe (Olive, Warm Gold, Off-white), Glassmorphism, large rounded corners.
- **Pages:** Home, Menu, Grocery, Reviews, Contact.
- **Interactions:**
  - Sticky navbar with background blur on scroll
  - Mobile hamburger menu navigation
  - Filterable Menu gallery with animation
  - Swipeable Reviews carousel (Swiper.js)
  - Scroll animations (AOS.js)
  - Floating WhatsApp and Scroll-to-Top buttons
- **Responsive:** Mobile-first approach, fully responsive on tablets and desktops.

## 📁 File Structure
- `index.html` - Homepage with highlights.
- `menu.html` - Filterable menu with dish cards.
- `grocery.html` - Grocery selection.
- `reviews.html` - Google reviews carousel.
- `contact.html` - Working contact/reservation form mapped to formspree and Google Maps embed.
- `css/style.css` - Full styling.
- `js/script.js` - Interactions logic.
- `images/` - Image directories (using high-quality Unsplash image placeholders in HTML directly according to requirement as dynamic image generation API met strict limits, giving you instant beautiful visualization).

## 🛠️ Deployment Instructions
1. Upload all the files inside `restaurant-website` folder to Netlify, Vercel, or GitHub Pages.
2. The site is statically rendered and requires NO build tools or backend.
3. For the contact form to receive emails, update the `action="https://formspree.io/f/placeholder"` inside `contact.html` to your own Formspree endpoint.

Enjoy your beautiful new site!
